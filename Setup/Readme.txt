Color Picker v1.1 (FREEWARE)
===============================
ColorPicker is small program which picks the color
value from selected screen pixel and converts it to
the RGB color values, calculates hexadecimal and
hexadecimal in HTML format (#xxxxxx) color values.

ColorPicker also can be used as simple calculator,
to convert RGB color values to HTML format. 

System requirements:
====================
Windows 95/98/ME/NT4/2000
